# חיפוש פונטי - שלב ב

בשלב ב עליכם לכתוב מימוש מלא של הפונקציה. 
המימוש צריך לעבור את כל הבדיקות - שלכם ושלנו.

אנחנו מריצים את הפקודה

<div dir='ltr'>

	bash grade

</div>

שמריצה את הפקודות:

<div dir='ltr'>

	make test
	./test

</div>

שימו לב: אין לשנות קבצים קיימים אלא רק להוסיף קבצים חדשים.
